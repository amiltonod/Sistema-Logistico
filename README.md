# 🚛 Sistema de Programação Logística

Sistema desenvolvido em Python para gerenciamento de coletas logísticas diretamente pelo terminal.

O projeto permite cadastrar, visualizar e remover programações de coleta de forma simples e organizada, utilizando armazenamento em arquivo JSON.

---

# 📌 Funcionalidades

✅ Cadastro de coletas  
✅ Visualização da programação completa  
✅ Remoção de coletas cadastradas  
✅ Armazenamento automático em JSON  
✅ Interface simples via terminal  
✅ Persistência de dados mesmo após fechar o sistema  

---

# 🛠️ Tecnologias utilizadas

- Python 3
- JSON
- PyCharm
- Git/GitHub

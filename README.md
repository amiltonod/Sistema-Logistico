# 🚛 Sistema de Programação Logística

Sistema logístico desenvolvido em Python para gerenciamento de coletas e fornecedores diretamente pelo terminal.

O projeto foi criado com foco em aprendizado prático de desenvolvimento backend, arquitetura modular e integração com banco de dados SQLite.

---

# git Funcionalidades

## 🚚 Módulo de Coletas

✅ Cadastro de coletas  
✅ Visualização da programação  
✅ Busca de coletas por cliente  
✅ Edição de coletas  
✅ Remoção de coletas  
✅ Estatísticas por prioridade  
✅ Persistência de dados  

---

## 🏢 Módulo de Fornecedores

✅ Cadastro de fornecedores  
✅ Listagem de fornecedores  
✅ Integração com SQLite  
✅ Estrutura modular separada

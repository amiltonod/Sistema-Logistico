# 🚛 Sistema de Programação Logística

Sistema desenvolvido em Python para gerenciamento logístico via terminal, utilizando arquitetura modular, SQLite e integração com API de CNPJ.

O projeto está evoluindo progressivamente como parte do meu aprendizado prático em desenvolvimento de software, arquitetura de sistemas e automação de processos logísticos.
